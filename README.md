### 과제 수행 사항
1. Github 개인 계정 / 첫번째 repository + README.md 파일 만들어보기(repository 이름: YSAL-rookie-1-본인이름)
2. 본인 학번으로 된 branch를 folk 받는다.

5. 본인 repository에 수정한 파일을 commit message와 함께 commit한다.
